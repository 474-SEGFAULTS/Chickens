Chickens

SinglePlayer Scene

Instructions

Welcome to Chickens! This is a fun single player game.

The game consisits of you, the chicken, running around a map trying to defeat different enemies that appear in the map.

Use the 'w' key to jump, the 'a' key to move left, the 'd' key to move right, the 's' key to go down, and the space bar to shoot.

The game consists of 5 rounds/levels to compete and each level gets harder.

Good luck Chicken, go defeat those enemies!
